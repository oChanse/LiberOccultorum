# Liber Occultorum

**Liber Occultorum — De Mysteriis et Doctrinis**  
Uma biblioteca digital dedicada ao estudo das tradições espirituais, filosóficas e esotéricas do mundo.

Este projeto nasceu como um espaço de investigação simbólica.  
Hoje ele reúne conteúdos sobre:

- Filosofia do Caos e Metafísica
- Religiões Antigas e Comparadas
- Judaísmo, Cristianismo e Islã
- Tradições Afro-diaspóricas
- Gnose, Hermetismo e Alquimia
- Demonologia Histórica e Goetia
- Psicologia Analítica (Jung)
- Simbolismo, Mitologia e Narrativa

Liber Occultorum não é um blog.  
Não é apenas um repositório de textos.  

É uma estrutura de estudo.

---

## Propósito

Este projeto existe para:

- Organizar conhecimento simbólico de forma acessível.
- Explorar tradições religiosas com respeito histórico.
- Analisar esoterismo sob perspectiva filosófica e psicológica.
- Criar uma base estruturada para estudo independente.

Ele é tanto um produto funcional quanto um laboratório intelectual.
